Jose Elera's Vim Configuration

This is my Vim setup for Web Development with HTML, CSS and Javascript. It is based on the .vimrc shared by Amix.dk

I've modified the following * Yi Zhao's Javascript syntax file to make the highlighting more appealing and easier to read to me. * A customized version of "ir_black" color scheme, I've named it "nazca" (in honor to a very nice place in Peru). It has a nice contrast.

Plugins Included

Even though I have all these plugins installed in my .vim directory, if you like any of them, please go to the vim script site, dowload it and vote for it.

These plugins are installed in the .vim\plugin

  • Align.vim : Align text, eqns, declarations, tables, etc.
  • Vim Color Picker for GTK : A simple color picker for VIM, based on GTK color chooser dialog.
  • Command-t.vim : Fast file navigation for VIM Ruby Interp Required.
  • ctags.vim : Display function name in the title bar and/or the status line.
  • hexHighlight.vim : Highlight hex codes in graphical Vim.
  • hgrev.vim : Display Mercurial revision info in the Vim statusline.
  • jsbeautify.vim : a javascript source code formatter.
  • matchit.vim : extended % matching for HTML, LaTeX, and many other languages.
  • MiniBufExpl.vim : Elegant buffer explorer - takes very little screen space.
  • PickAColor.vim : Inserts and edits color codes using a color name or a graphic color chooser.
  • taglist : Source code browser (supports C/C++, java, perl, python, tcl, sql, php, etc)
  • TaskList : Eclipse like task list, like FIXME, TODO and XXX (or a custom list)
  • Rainbow Parenthesis : Highlight matching parens in a rainbow of colors.
  • YankRing : Maintains a history of previous yanks, changes and deletes.

These following plugins are installed as git submodules in ~/.vim/bundle with the Pathogen Plugin

  • ack.vim : Plugin for the Perl module / CLI script 'ack' replacement for grep
  • auto_mkdir : Allows you to save files into directories that do not exist yet.
  • autoclose : Inserts matching bracket, paren, brace or quote.
  • gundo : Graph your Vim undo tree in style.
  • jslint.vim : Check JavaScript files for errors with JSLint.
  • neocomplcache : Ultimate auto completion system for Vim.
  • NERDCommenter : A plugin that allows for easy commenting of code for many filetypes.
  • NERDtree : A tree explorer plugin for navigating the filesystem.
  • rvm : Adds RVM integration to Vim. Which active Ruby is being used shown in the status line.
  • syntastic : Automatic syntax checking.
  • Endwise : Wisely add "end" in ruby, endfunction/endif/more in vim script, etc.
  • Fugitive : A Git wrapper so awesome, it should be illegal.
  • Vim-Git : Git runtime files .
  • Indent-guides : A plugin for visually displaying indent levels in Vim.
  • ragtag : A set of mappings for HTML, XML, PHP, ASP, eRuby, JSP, and more (formerly allml).
  • rails.vim : Ruby on Rails: easy file navigation, enhanced syntax highlighting, and more.
  • vim-rdoc ; Syntax highlight for Ruby Documentation.
  • vim-ruby : Vim/Ruby Configuration Files.
  • Surround : Delete/change/add parentheses/quotes/XML-tags/much more with ease.
  • xmledit : A filetype plugin to help edit XML, HTML, and SGML documents.
  • zencoding : vim plugins for HTML and CSS hi-speed coding.
